# 信用卡盗刷处理流程

## 一、发现异常交易

持卡人发现异常交易后应立即采取以下措施：
1. 立即拨打24小时客服热线 400-888-8888 挂失卡片
2. 登录手机银行APP冻结卡片（临时止付）
3. 记录异常交易的时间、金额、商户信息

## 二、盗刷申报

### 申报时效
- 挂失前48小时内的盗刷，银行承担最高5万元
- 白金卡及以上挂失前72小时内，银行承担最高20万元
- 超过时效的盗刷损失由持卡人承担

### 申报材料
1. 盗刷争议申报表（手机银行APP在线填写）
2. 身份证正反面照片
3. 信用卡正反面照片
4. 报警回执（金额超过5000元需提供）
5. 交易当时不在场的证明材料（如出行记录、消费凭证等）

## 三、调查处理

| 阶段 | 时限 | 说明 |
|------|------|------|
| 初审 | 3个工作日 | 审核材料完整性 |
| 调查 | 30个自然日 | 调取交易记录、商户信息 |
| 裁定 | 5个工作日 | 出具调查结论 |
| 退款 | 3个工作日 | 认定盗刷后退还争议金额 |

## 四、防范建议

- 开通交易提醒（短信/APP推送）
- 设置境外交易开关，不使用时关闭
- 不在公共WiFi下进行支付操作
- 定期检查账单，及时发现异常
