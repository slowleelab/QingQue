# 信用卡还款方式说明

## 还款渠道

### 1. 自动还款（推荐）
- 绑定本行借记卡，每月还款日自动扣款
- 支持全额还款或最低还款两种模式
- 需确保借记卡余额充足
- 设置方式：手机银行APP → 信用卡 → 自动还款设置

### 2. 手机银行还款
- 本行借记卡转账还款：实时到账，免手续费
- 跨行还款：1-2个工作日到账，免手续费

### 3. 网银还款
- 登录个人网上银行，选择信用卡还款
- 本行借记卡还款实时到账

### 4. ATM还款
- 本行ATM存入现金：实时到账
- 本行ATM转账还款：实时到账

### 5. 柜台还款
- 现金还款：实时到账
- 转账还款：实时到账

### 6. 第三方平台还款
- 支付宝/微信/云闪付均可还款
- 到账时间：1-2个工作日
- 注意：需提前2天操作以确保还款日到账

## 还款注意事项

- 还款日当日24:00前还款均视为按时还款
- 跨行转账需预留1-2个工作日
- 外币消费需购汇还款或开通自动购汇
- 还款金额以银行入账金额为准
