# 信用卡年费减免政策

## 消费达标减免

### 普卡/金卡
- 首年免年费：卡片激活即享受，无需消费达标
- 次年减免：当年累计消费满6次（含线上线下），不限金额
- 消费次数计算周期：上一年度1月1日至12月31日

### 白金卡
- 首年免年费：卡片激活即享受
- 次年减免：满足以下任一条件
  - 当年累计消费满12次
  - 当年累计消费金额满30,000元
  - 当年积分消费满100,000积分抵扣

### 钻石卡
- 首年免年费：卡片激活即享受
- 次年减免：当年累计消费金额满100,000元

## 特殊减免政策

| 客户类型 | 减免政策 | 说明 |
|---------|---------|------|
| 新客户首卡 | 首两年免年费 | 首次办理本行信用卡客户 |
| 薪资卡客户 | 终身免年费（普卡/金卡） | 本行代发工资客户 |
| 公务卡 | 终身免年费 | 财政预算单位公务消费 |
| 退休人员 | 年费5折 | 60岁以上持卡人 |
| 军人/消防 | 年费5折 | 现役军人及消防救援人员 |

## 不计入消费达标次数的交易

1. 信用卡取现
2. 信用卡转账
3. 年费、滞纳金、超限费等费用
4. 退款交易（撤销原消费次数）
5. 公益捐款
6. 预授权完成前的冻结交易
