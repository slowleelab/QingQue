# 信用卡分期常见问题

## Q1：分期有哪些类型？

- 消费分期：单笔消费满500元即可申请，支持3/6/12/24期
- 账单分期：账单金额满800元即可申请，支持3/6/12/24/36期
- 现金分期：将信用额度转入借记卡，支持3/6/12/24期

## Q2：分期手续费怎么收？

分期手续费按月收取，费率如下：

| 期数 | 月费率 |
|------|--------|
| 3期 | 0.70% |
| 6期 | 0.60% |
| 12期 | 0.55% |
| 24期 | 0.60% |
| 36期 | 0.65% |

## Q3：分期可以提前还款吗？

可以提前还款，但需一次性偿还剩余本金及剩余期数的手续费。不支持部分提前还款。

## Q4：分期占用额度吗？

是的，分期金额全额占用信用额度。每还一期，释放对应本金部分的额度。

## Q5：分期申请有条件吗？

- 信用卡状态正常，无逾期记录
- 消费分期需在消费后至账单日前申请
- 账单分期需在账单日后至还款日前2天申请
