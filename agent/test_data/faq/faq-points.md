# 信用卡积分常见问题

## Q1：积分如何获取？

每消费1元人民币积1分（普卡/金卡），白金卡消费1元积1.5分，钻石卡消费1元积2分。境外消费双倍积分。

## Q2：积分有效期是多久？

- 普卡/金卡：积分有效期为2年
- 白金卡：积分有效期为3年
- 钻石卡：积分永久有效

## Q3：积分可以兑换什么？

积分可兑换以下项目：
- 航空里程：25积分=1里程
- 商城商品：详见积分商城
- 话费充值：500积分=10元话费
- 年费抵扣：按卡种对应年费等值积分抵扣

## Q4：哪些消费不计积分？

以下消费类型不计积分：
- 信用卡取现及手续费
- 信用卡年费、滞纳金、超限费
- 公益捐款
- 房地产类、汽车类商户消费
- 批发类商户消费

## Q5：积分可以转让吗？

积分不可转让给其他持卡人，但主卡持卡人可将附属卡积分合并至主卡账户使用。
