# 信用卡年费常见问题

## Q1：信用卡年费是多少？

不同卡种的年费标准如下：

| 卡种 | 年费标准 |
|------|---------|
| 普卡 | 100元/年 |
| 金卡 | 300元/年 |
| 白金卡 | 1000元/年 |
| 钻石卡 | 3000元/年 |

## Q2：如何免年费？

- 普卡/金卡：首年免年费，当年消费满6次免次年年费
- 白金卡：首年免年费，当年消费满12次或累计消费满3万元免次年年费
- 钻石卡：首年免年费，当年累计消费满10万元免次年年费

## Q3：年费什么时候收取？

年费在卡片激活后的首个账单日收取，此后每年同一账单日收取。如果满足免年费条件，系统将自动减免，无需额外申请。

## Q4：年费可以退还吗？

年费一经收取，如已满足减免条件，可在账单日后3个工作日内自动退还。超过3个工作日需致电客服热线申请人工退还。

## Q5：附属卡有年费吗？

附属卡年费为主卡年费的50%。附属卡同样享受消费达标免年费政策，消费次数与主卡合并计算。
