# 信用卡积分有效期及清零规则

## 积分有效期

| 卡种 | 有效期 | 计算方式 |
|------|--------|---------|
| 普卡 | 2年 | 自获得之日起，按自然年计算 |
| 金卡 | 2年 | 自获得之日起，按自然年计算 |
| 白金卡 | 3年 | 自获得之日起，按自然年计算 |
| 钻石卡 | 永久 | 无过期限制 |

## 清零规则

- 积分在到期月的最后一天清零
- 清零前1个月、前1周、前1天分别发送提醒短信
- 清零顺序：先获得的积分先清零（FIFO）
- 年度清零时间：每年12月31日23:59:59

## 积分保留与转移

| 场景 | 处理方式 |
|------|---------|
| 卡片升级 | 积分保留，有效期按新卡种重新计算 |
| 卡片降级 | 积分保留，有效期不变，积分倍数按新卡种执行 |
| 卡片到期换卡 | 积分自动转移至新卡 |
| 销卡 | 积分在销卡前需兑换完毕，销卡后积分清零 |
| 销户 | 所有卡片积分清零，不可恢复 |

## 延长积分有效期

- 参与指定促销活动可延长6个月有效期
- 白金卡持卡人可用5000积分兑换1年延期
- 钻石卡积分永久有效，无需延期
