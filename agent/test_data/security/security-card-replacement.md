# 信用卡挂失补卡流程

## 挂失方式

| 方式 | 办理时间 | 费用 |
|------|---------|------|
| 客服热线 400-888-8888 | 24小时 | 普卡/金卡50元，白金及以上免费 |
| 手机银行APP | 24小时 | 同上 |
| 网点柜台 | 工作日9:00-17:00 | 同上 |

## 补卡流程

1. 挂失确认：挂失后卡片立即冻结，所有交易被拒绝
2. 申请补卡：挂失时可同时申请补卡
3. 卡片制作：7-10个工作日（同卡号），3-5个工作日（换号）
4. 邮寄送达：EMS免费邮寄至账单地址
5. 激活使用：收到新卡后致电客服或APP激活

## 特殊情况

### 卡片损坏换卡
- 磁条损坏/芯片故障可免费换卡
- 换卡不换号，7-10个工作日

### 有效期到期换卡
- 到期前1个月自动寄送新卡
- 新卡卡号不变，额度不变
- 收到新卡后激活，旧卡自动失效

### 加急补卡
- 仅限白金卡及以上
- 2个工作日制作完成
- 加急费50元
- 支持网点自取

## 安全提示

- 挂失后原卡所有绑定关系保留（支付宝/微信等）
- 新卡号变更需重新绑定第三方支付
- 收到新卡后请立即签字并激活
- 如未收到新卡，请及时联系客服查询快递状态
