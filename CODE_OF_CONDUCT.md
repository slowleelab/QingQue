# 贡献者公约行为准则

## 我们的承诺

作为成员、贡献者和领导者，我们承诺让每个人——无论年龄、体型、显性或隐性残疾、族裔、性别认同与表达、经验水平、教育程度、社会经济地位、国籍、个人形象、种族、宗教或性认同与性取向如何——都能在社区中获得无骚扰的体验。

我们承诺以有助于营造开放、友善、多元、包容且健康的社区的方式行事和互动。

## 我们的标准

有助于营造积极环境的行为包括：

- 对他人表现出同理心与善意
- 尊重不同的意见、观点和经验
- 给予并优雅地接受建设性反馈
- 承担责任，向受影响的人道歉，并从错误中学习
- 关注不仅对个人、更是对整个社区最有利的事

不可接受的行为包括：

- 使用性化的语言或 imagery，以及任何形式的性关注或挑逗
- 挑衅、侮辱或贬损性评论，以及人身或政治攻击
- 公开或私下骚扰
- 未经明确许可，公布他人私人信息（如住址或邮箱）
- 其他在专业环境中可被合理认定为不当的行为

## 执行责任

社区领导者有责任澄清和执行我们可接受行为的标准，并将对任何其认为不当、威胁、冒犯或有害的行为采取适当且公正的纠正措施。

社区领导者有权利和责任删除、编辑或拒绝不符合本行为准则的评论、提交、代码、wiki 编辑、Issue 和其他贡献，并在适当时说明管理决定的理由。

## 适用范围

本行为准则适用于所有社区空间，也适用于个人在公共空间正式代表社区时（如使用官方邮箱、通过官方社交媒体账号发布，或在线上/线下活动中担任指定代表）。

## 执行

辱骂、骚扰或其他不可接受的行为可向负责执行的社区领导者报告（通过仓库 Issue 中提供的联系方式或私密渠道）。所有投诉都将得到及时、公正的审查和调查。

所有社区领导者有义务尊重任何事件报告者的隐私和安全。

## 执行指南

社区领导者将遵循以下社区影响指南，来确定违反本行为准则行为的后果：

### 1. 纠正
**社区影响**：使用不当语言或其他被认为不专业或不受欢迎的行为。
**后果**：社区领导者私下书面警告，说明违规性质并解释为何不当。可能要求公开道歉。

### 2. 警告
**社区影响**：通过单一事件或一系列行为造成的违规。
**后果**：警告并说明持续行为的后果。在规定期限内不得与相关人员互动，包括主动与执行准则者互动。违反这些条款可能导致临时或永久封禁。

### 3. 临时封禁
**社区影响**：严重违反社区标准，包括持续的不当行为。
**后果**：在规定期限内临时禁止与社区进行任何形式的互动或公开交流。违反这些条款可能导致永久封禁。

### 4. 永久封禁
**社区影响**：表现出违反社区标准的模式，包括持续的不当行为、骚扰个人，或对某类个体的攻击或贬低。
**后果**：永久禁止在社区内进行任何形式的公开互动。
